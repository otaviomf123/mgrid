mgrid Documentation
===================

.. image:: _static/example_variable_resolution.png
   :alt: Variable Resolution Grid Example
   :align: center
   :width: 80%

**mgrid** is a Python package for generating variable-resolution spherical meshes
for MPAS-based atmospheric models (MPAS, MONAN, and compatible systems).
